# ATM Simulator

This is a simple ATM simulator application built using C#. It allows users to perform basic banking operations such as checking their balance, depositing money, and withdrawing money.

It was made by Taddycooper aka myself as a fun project because I was bored one day lol.

The Currency used in this simulator is not real. but it is supposed to simulate USD.

# Features

- Check Balance
- Withdraw Money
- Deposit Money
